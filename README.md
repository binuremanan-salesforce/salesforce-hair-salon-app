# salesforce hair salon app
A complete Salesforce Administrator portfolio project demonstrating custom objects, flows,process builder,data and metadata security, validation rules, reports, dashboards, and approval processes.

## Overview

This project is a complete Salesforce Administrator portfolio application designed for a Hair Salon business. It demonstrates practical Salesforce Administration skills including customisation, automation, security, reporting, and data impoting and exporting.
The application helps salon staff manage appointments, services, payments, products, and customer interactions efficiently.

## Business Requirements

Manage customer appointments
Track salon services
Process payments
Maintain product inventory
Automate appointment workflows
Generate management reports
Control user access securely
